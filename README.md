# Soft-Ui-Dashboard
Soft-Ui-Dashboard

This is a modern and responsive Soft UI Dashboard built with HTML, CSS, and JavaScript, designed for clean data visualization and user-friendly admin interfaces. It features a sleek layout, reusable components, and mobile-responsive design for real-world projects.

Steps to Run the Dashboard

Extract the ZIP file
Right-click the ZIP → Extract All.

Open in VS Code
Open VS Code → File → Open Folder → select the extracted folder.

Run the dashboard

Recommended: Right-click index.html → Open with Live Server

Or: Double-click index.html to open it in a browser.

✅ No backend or server setup required — it runs directly in the browser.
